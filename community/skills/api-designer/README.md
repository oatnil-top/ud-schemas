# api-designer

> Design RESTful APIs following best practices and conventions

**Category**: Development
**Author**: UnDercontrol
**Tags**: `development`, `api`, `design`, `architecture`

## What it does

This skill helps design RESTful APIs with consistent conventions: resource-oriented URLs, proper HTTP method usage, standardized response structures, pagination, filtering, and error handling.

## Install

```bash
ud market install skill api-designer
```

Or manually:
```bash
curl -sL https://raw.githubusercontent.com/oatnil-top/ud-schemas/main/community/skills/api-designer/skill.yaml | ud apply -f -
```

## Usage

```bash
ud prompt api-designer
```
