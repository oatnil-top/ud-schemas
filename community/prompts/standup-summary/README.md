# standup-summary

> Summarize a task for daily standup updates

**Category**: Project Management
**Author**: UnDercontrol
**Tags**: `standup`, `summary`, `daily`

## What it does

A workspace prompt that generates a concise standup summary from a task's context. Outputs three sections: Done, Doing, and Blockers.

## Install

```bash
ud market install prompt standup-summary
```

Or manually:
```bash
curl -sL https://raw.githubusercontent.com/oatnil-top/ud-schemas/main/community/prompts/standup-summary/prompt.yaml | ud apply -f -
```
